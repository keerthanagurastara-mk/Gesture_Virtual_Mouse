# Gesture Virtual Mouse

## 🚀 Overview

A computer vision-based virtual mouse that allows users to control the system cursor using hand gestures.

## ✨ Features

* Hand tracking using OpenCV & MediaPipe
* Gesture-based mouse control
* Smooth cursor movement
* Modular architecture

## 🖥️ Demo

![Demo](assets/demo.gif)

## ⚙️ Installation

```bash
pip install -r requirements.txt
```

## ▶️ Usage

```bash
python main.py
```

## 📁 Project Structure

* `modules/` → Core functionality
* `utils/` → Helper functions
* `config/` → Settings/configurations
* `tests/` → Unit tests

## 🧠 Future Improvements

* ML-based gesture recognition
* Performance optimization
* GUI interface

## 📌 Tech Stack

* Python
* OpenCV
* MediaPipe
